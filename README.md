💰 FinAgent – Finance Manager

FinAgent is a modern, responsive finance management web application built using React. It helps users track expenses, manage budgets, and monitor financial activity through a clean and professional interface.

🚀 Features
Expense tracking (add, edit, delete)
Budget management (user-defined)
Savings tracking
Live dashboard with real-time updates
Expense filtering and categorization
Smart overspending alerts
Local data persistence using browser storage
Fully responsive design for all devices
🛠️ Tech Stack
React (Functional Components)
Context API (Global State Management)
JavaScript (ES6+)
CSS (Custom styling & animations)
LocalStorage (Data persistence)
